Within this repository are all components of my class project proposal.

Product Name

Crossroads

Team Members

Rowan Bailey (solo)

Application Overview (end-user perspective)

Crossroads is a choose-your-own-adventure game. Players read a short scene, pick from 2–3 choices, and branch to new scenes with unique outcomes.

Technology Overview

  • Languages: HTML5, CSS3, JavaScript
  • Libraries: None (vanilla JS)
  • Platforms: Modern desktop/mobile browsers
  • Hosting: GitHub Pages
  • Data: JSON scene graph + localStorage

Proposal Last Updated

November 2, 2025

Feature List (prioritized)

  • Core branching engine
  • Multiple endings with restart and quick “back one step”
  • Minimal, readable UI
  • Basic analytics counter (endings reached, choices made) displayed locally

Future Feature List

  • Sound effects / music toggle
  • Timed choices and hidden variables
  • Achievements

Basic Technical Features

  • HTML structure and responsive CSS
  • Modular JS

Advanced Technical Features

  • Simple event bus to track metrics (choice frequency, endings reached)
